C++常量

const

关键字const叫做限定符，它限定了声明的含义

编译器不允许再修改该常量的值，g++会提示程序试图给一个只读变量赋值

const比define更好，原因：

<1> 明确指明类型

<2> C++的作用域规则将该定义限制在特定的函数或文件中

被const指定了的函数返回值表示为常量，不能再当作变量被赋值（也就是不能充当左值）

C++不允const T* 转换为 T*，除非使用const_cast显示转除const限定符

<1> 有const修饰的成员函数(const放在函数参数列表后面)，只能读取数据成员，不能修改数据成员；没有const修饰的成员函数，对数据成员是可读可写的

<2> 常量对象（即const对象）可以调用const成员函数，而不能调用非const修饰的成员函数

<3> 非常量对象既可以调用常量成员函数，也可以调用非常量成员函数

文字常量在data区

常量

C++常量分为6种

整型常量 实型常量 字符型常量 字符串常量 符号常量 逻辑型常量

牛客刷题笔记

多态

指针和引用都可以实现多态

对于单继承，如果派生类没有重写基类的函数，则使用最近祖先中的定义

对于多继承，如果派生类没有重写基类的函数，会产生二义性，可以用作用域解析运算符表明使用哪一个基类

int (p)[N] 二维数组
int p[N] 一维指针数组

指针大小由机器字长决定

#pragma pack(2) //2字节对齐，默认4字节对齐（32位），默认对齐与机器字长有关（内存读取方式也有关）

enum 类型默认4字节，溢出时强转为8字节

enum 也可显式表明长度 uint8_t uint16_t uint32_t uint64_t

类的内存大小

空的类是会占用内存空间的，而且大小是1，原因是C++要求每个实例在内存中都有独一无二的地址。

（一）类内部的成员变量：
普通的变量：是要占用内存的，但是要注意对齐原则（这点和struct类型很相似）。
static修饰的静态变量：不占用内容，原因是编译器将其放在全局变量区。

（二）类内部的成员函数：
普通函数：不占用内存。
虚函数：要占用4个字节（32位），用来指定虚函数的虚拟函数表的入口地址。所以一个类的虚函数所占用的地址是不变的，和虚函数的个数是没有关系的

只有一个变量时，char 1, int 4, double 8 和变量长度一致

当有多个变量时，对齐是8B（64）位

<1> char + int 占8B |-___—-| //-表示占据 _表示空闲

<2> char + double 占16B |-_______| |——–|

C预处理

预处理的概念

预处理主要在编译之前处理程序中的特殊行

预处理指令包括： 宏定义、文件包含和条件编译

c程序 -> 预处理器 -> 修改后的c程序 -> 编译器 -> 目标代码

预处理器是一个小软件，在编译前处理c程序

预处理操作

<1> 宏定义。 #define指令定义一个宏， #undef指令删除一个宏

<2> 文件包含。 #include指令将一个指定文件内容包含到程序中

<3> 条件编译。 #if、#ifdef、#ifndef、#elif、#else和#endif指令根据测试条件来确定是否将一段程序文本包含到程序中

规则

<1> 以#开始

<2> 换行符作为结尾，’'可延续到下一行 (编译器会忽略行尾的换行符，起到续行作用)

<3> 可间隔任意空格、制表符，可放在程序任意位置，一般放在文件开始位置，可与注释同行

#define

#define 标识符 替换文本 //符号常量

#define 宏名(x1,x2,..) 替换列表 //带参数的宏定义

参数带上括号程序更安全

带参数的宏定义不是函数

宏定义没有类型检查

#include

格式1：#include “文件名” 优先查找工作目录，其次查找预先指定的编译器和系统目录。

格式2：#include <文件名> 在预先指定的编译器和系统目录查找

编译多文件时，无需编译.h文件，因为编译.c文件时会自动编译头文件的内容

编译生成一个包含目标代码的文件，称为目标文件，windows系统中扩展名为.obj

链接器把目标文件和库函数代码结合在一起生成可执行程序

链接器可以解决外部引用问题(extern 外部函数)

条件编译

条件编译可以用来保护头文件(文件嵌套无意间包含一个头文件多次)

源文件包含函数定义和函数调用

头文件包含类型定义、函数原型声明等

C++学习笔记

4.5 共用体 （union）

用法和类相同

长度为最大成员的长度 共用同一块内存

匿名共用体

类似于同一起始地址有多个成员，但同时只能有一个成员活跃

常用于操作系统数据结构和硬件数据结构 节约内存

4.6 枚举 （enum）

创建符号常量的方式，可代替const

为了获取最大的移植性，将非enum值赋给enum变量视为error

enum为整型，可自动转为int

int要强制类型转换才能变成enum变量的值

可选择性赋值，默认第一个为0，以后每个枚举量增加1

可以创建多个相同值的枚举量

取址范围

上限为 最大值所在2的幂次的上限减1，120上限为127

下限为 最小值大于0，则取零；否则与上限相同，取负值，-6下限为-7

重载、覆盖和隐藏

重载

同一个类中的不同函数

<1> 相同的范围（在同一个类中）

<2> 函数名相同

<3> 参数不同

覆盖

派生类函数覆盖基类函数

<1> 不同范围（基类和派生类）

<2> 函数名相同

<3> 参数相同

<4> 基类函数带virtual

隐藏

派生类函数屏蔽了同名的基类函数

<1> 同名，参数不同，有无virtual基类函数都被隐藏

<2> 同名，同参，无virtual时基类函数被隐藏
